# Network Ruler v2.0.0

Network Ruler v2.0.0 is the Windows-first rebuild focused on safer operations,
clearer command structure, and a polished PySide desktop shell.

## Highlights

- Fast root commands for common process, network, DNS, and bandwidth tasks.
- Structured `process`, `network`, `monitor`, `profile`, `alias`, and `config`
  command groups.
- Preview-first write actions with dry-run support and explicit confirmations.
- Seven-screen desktop GUI: Dashboard, Processes, Network, Monitor, Profiles,
  Logs, and Settings.
- User-scoped config, cache, data, and log paths outside the repository.
- Runtime logging through the Network Ruler log file.
- GitHub Actions workflows for pytest and ruff.

## Verification

- `pytest`: 107 passing tests.
- `ruff check .`: all checks passed.

## Notes

Network Ruler v2.0.0 is intentionally Windows-first. Linux and macOS support are
not release goals for this milestone.
